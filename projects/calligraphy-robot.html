Final Year Project: Chinese Calligraphy Robot

Sept 2024 - Nov 2025

Technologies: Python, Tkinter, Threading, Hermite Curves, YOLOv8, ChatGPT API

Overview

This project combines robotics, computer vision, and AI to create a system that can write authentic Chinese calligraphy using a robotic arm. The key innovation is a 3D Hermite curve-based path planning algorithm that controls the brush height to modulate stroke thickness - a crucial aspect of traditional calligraphy. The system accepts user prompts in natural language, uses ChatGPT to suggest appropriate four-character idioms, and then generates the corresponding brush trajectories.

  • Brush height is modulated via largest fitting circles algorithm - lower brush = thicker strokes.
  • Supports 2D (fixed height) and 3D (variable height) modes for comparison.

How It Works (Step by Step)

  1. User Input: User types a theme (e.g., "mother's day") into the Tkinter UI.
  2. LLM Suggestion: ChatGPT returns a four-character idiom (e.g., ÊÅ©ÈáçÂ¶ÇÂ±±).
  3. Fetch Strokes: System retrieves 2D control points for each character from the Make Me a Hanzi dataset.
  4. 3D Control Points: For each stroke, calculate brush height using largest fitting circles - thicker parts press down.
  5. Trajectory Generation: Connect points with Hermite curves (using Catmull-Rom to compute tangents).
  6. Robot Control: Send JSON commands to the 5-DoF robotic arm (via serial) with x, y, z, angle, and claw state.
  7. Real-time Monitoring: Camera runs YOLOv8 to detect brush bending; if detected, emergency stop engages.
